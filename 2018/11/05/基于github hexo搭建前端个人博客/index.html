基于github hexo搭建前端个人博客

2018-11-05 PV:

right! this is you want

i`ll show you how to set up a free blog

作为一名开发者，特别是面试的时候，多多少少都会被涉及到一个问题：你是否有写博客？或者你是否有属于自己的博客？

几年工作下来，自己面试了好多家，也面试了好多人，大部分求职者对于这个话题都是哑口无言，随着互联网不断壮大，社区多元化，很多人才流动，已由传统的HR通过简历搜罗人才，变成社区里面寻找真正有墨水的人才，直观的体现，就是看你在某一领域的活跃度，博客就是最好的体现。对于工作而言，经常写博客，不仅仅是对自己阶段性知识的一个总结积累。更是同行交流一种比较直接的手段。

but.每个地区对与博客的重视程度不一样，这是由于地区的科技发展水平，以及人文所决定的，之前在厦门待过一段时间，基本上组里每个人都有自己的博客，他们乐意去做这些事情，也乐于去做分享，经常还会组织园区内公司开展交流活动，彼此都会先将自己的技术观点以博文的方式做记录，这是一种氛围，久而久之，作者本身对博文对博文对质量要求越来越高，同时通过哦这种途径让更多的人认识到你，一举两得。

所以，作为有追求的我们，2019年到来之处，是时候搭建一个属于自己的博客了。

如果你还不知道你为什么需要做一个博客，推荐你看我为什么要写博客

  1. 必须耐得住折腾与寂寞。

关于github

github的优点

  • github是基于git实现的代码托管，git可能是目前最好用的版本控制系统
  • github是免费的
  • github是程序员社交最广的平台，有利于别人发现你。

什么是github pages

github pages可以被认为是用户编写的、托管在github上的静态网页，可以绑定你想要的域名（当然前提是你得花钱去买你想要的域名，如果你厌倦了github本身提供的域名），还可以为你提供一个免费的服务器，免去你大服务器和建数据库的麻烦。

部署到github pages上。

那么，现在我们已经将博客搭建起来了。但是我们只能通过本地连接查看我们的博客，接下来我们熬做的就是将本地博客发布到服务器上。让别人也能看到我们的博客。github pages就是来帮助我们完成这样的事情。但是github pages界面的代码时在github上的，所以你必须到github上创建自己的账户。
